auc曲线计算：

​ AUC指标也是机器学习、统计学中关于分类问题一个常见的指标，这里只记录在python中如何计算AUC，至于AUC的细节问题，专门开一个博客讲解。

​ 在python，常用sklearn包来计算AUC，那么如何计算呢？首先上代码，这里只分析二分类：

from sklearn.datasets import load_breast_cancer
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import roc_auc_score
X, y = load_breast_cancer(return_X_y=True)
clf = LogisticRegression(solver="liblinear", random_state=0).fit(X, y)
roc_auc_score(y, clf.predict_proba(X)[:, 1])
roc_auc_score(y, clf.decision_function(X))

我们可以看以下X、y、clf.predict_proba(X)的shape，以及clf.predict_proba(X)是什么：

我们可以看到一组X、y是由一个30维的向量与一个label构成，而预测的结果是一个二维向量，在计算roc_auc_score中，是取[:, 1]，也就是预测为正例的概率。

对于二分类图像DeepFakes检测中，判断一张图片是否为fake，那么正例就是fake，由此可以计算AUC指标。

crossEntropyloss回顾：

这个函数是在pytorch中常用交叉熵损失函数，其用法如下：

output = model(data) # 注意，这里的output是Linear之后的output，并无激活函数
loss = nn.CrossEntropyLoss(output,label) # 注意，这里的label是类别，而不是one-hot编码

那么这个函数具体做了什么，主要包含两个部分：nn.LogSoftmax()和nn.NLLLoss()，具体来说做了下面三件事：

1、Softmax后的数值都在0~1之间，且它们之和为1，所以ln之后值域是负无穷到0。

2、然后将Softmax之后的结果取log，将乘法改成加法减少计算量，同时保障函数的单调性 。

3、NLLLoss的结果就是把上面的输出与Label对应的那个值拿出来，去掉负号，再求均值。

在这里不需要知道太多的细节，关于softmax与交叉熵损失，我会再起一篇博客仔细记录。

pytorch自定义数据增强方式：

这个是我觉得很有用的一个东西，在pytorch中自带了一个数据增强方式torchvision，但是这个库中只有一些常见的数据增强，甚至都没有Gauss Blur，链接：https://github.com/pytorch/vision

这里介绍两种常用的数据增强库：

1.imgaug，链接：https://github.com/aleju/imgaug

2.albumentations，链接：https://albumentations.readthedocs.io

值得一提的是，第二种方法可以自定义transform，很好用，封装了ToTensor，而第一种我尝试结合torchvision最终失败。

如何在torchvision自定义一个数据增强函数，可以采用如下代码：

class FUN(object):
    def __init__(self, length):
        self.length = length
        
    def __call__(self, img):
        # 这里的参数只能是img，最终返回img
        # 这里要注意是什么时候调用该函数，如果在ToTensor之后，由于其本身带有维度转换，因此要注意提取
        return img

最后调用方式为：

transform_train = transforms.Compose([
    transforms.Resize((224,224)),
    transforms.ToTensor(),
    FUN(length=16)
])
